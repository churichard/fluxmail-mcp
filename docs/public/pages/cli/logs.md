`fluxmail logs`

Show recent local log entries

## Usage

```bash
fluxmail logs [options]
```

## Options

| Option | Required | Details | Default |
| --- | --- | --- | --- |
| `--tail <count>` | No | Number of entries to show, from 1 through 1000 | `100` |
| `--level <level>` | No | Minimum level: info, warn, or error | `info` |
| `--json` | No | Print each entry as JSON | None |
