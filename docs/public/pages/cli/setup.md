`fluxmail setup`

Create the first local administrator or claim a migrated administrator

## Usage

```bash
fluxmail setup [options]
```

## Options

| Option | Required | Details | Default |
| --- | --- | --- | --- |
| `--name <name>` | No | Administrator name | None |
| `--email <email>` | No | Administrator email | None |
| `--existing-admin <member>` | No | Existing administrator id or email for a migrated instance | None |
