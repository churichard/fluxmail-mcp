`fluxmail apikey accounts`

Replace or clear an API key mailbox allowlist

## Usage

```bash
fluxmail apikey accounts <keyId> [options]
```

## Arguments

| Name | Required | Details | Default |
| --- | --- | --- | --- |
| `keyId` | Yes | None | None |

## Options

| Option | Required | Details | Default |
| --- | --- | --- | --- |
| `--account <account>` | No | Replace the allowlist with this mailbox; repeat as needed | None |
| `--all-accounts` | No | Clear the allowlist | None |
