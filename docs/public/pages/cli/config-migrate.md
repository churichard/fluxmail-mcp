`fluxmail config migrate`

Import settings from an env file without deleting it

## Usage

```bash
fluxmail config migrate [options]
```

## Options

| Option | Required | Details | Default |
| --- | --- | --- | --- |
| `--from <env-file>` | Yes | Read recognized settings from this env file | None |
| `--dry-run` | No | Show recognized setting names without changing Fluxmail | None |
