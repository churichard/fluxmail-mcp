`fluxmail oauth configure google`

Configure a custom Google OAuth application

## Usage

```bash
fluxmail oauth configure google [options]
```

## Options

| Option | Required | Details | Default |
| --- | --- | --- | --- |
| `--client-id <id>` | Yes | Google OAuth client ID | None |
| `--client-secret-file <path>` | No | Read the client secret from a file, or use - for stdin | None |
